## セクション詳細

### 0. ヘッダー `.header`

**構成要素：**
- ロゴ（買取コンシェルジュ）
- PC：右側に電話ボタン「無料査定はこちら」＋ LINEボタン
- SP：ハンバーガーメニュー

**PC / SP差分：**
- PC：ロゴ左 ＋ CTA右の横並び
- SP：ロゴ左 ＋ ハンバーガー右

**BEM構成案：**
```
.header
  .header__inner
    .header__logo
    .header__nav（PC）
    .header__cta
      .header__cta-phone
      .header__cta-line
    .header__hamburger（SP）
```

### 1. ファーストビュー `.hero`

**構成要素：**
- 古物商番号テキスト
- メインコピー「山梨県で出張買取をご検討の方へ」
- サブコピー「金・ROLEX・ブランドを丁寧に査定します」
- 補足テキスト「訪問するのは当社スタッフ 買取コンシェルジュです。」
- バッジ2つ「出張費・査定料0円」「強引な営業一切なし」
- CTA「無料査定を依頼する」（電話アイコン付き）
- CTA「LINEで写真査定」（LINEアイコン付き）
- 背景画像（スタッフ2名の写真）

**PC / SP差分：**
- PC：左にテキスト＋CTA、右に人物写真。CTAは縦並び（右寄せ）
- SP：背景画像の上にテキスト＋バッジ＋CTA。全要素縦積み

**BEM構成案：**
```
.hero
  .hero__inner
    .hero__content
      .hero__license（古物商番号）
      .hero__title
      .hero__subtitle
      .hero__description
      .hero__badges
        .hero__badge
        .hero__badge
    .hero__cta
      .hero__cta-phone
      .hero__cta-line
    .hero__image（PC表示用）
```

**GTM計測候補：**
- `data-gtm="hero-cta-phone"` 無料査定を依頼する
- `data-gtm="hero-cta-line"` LINEで写真査定

- 理由その1：正装での訪問

```

```
